# Stopwatch and Timer App

A clean, responsive, and minimalist React application featuring a Stopwatch and a Countdown Timer. Built using React, TypeScript, and Tailwind CSS.

## Features

* **Stopwatch:** High precision timer tracking hours, minutes, seconds, and centiseconds.
* **Timer:** Custom countdown timer where users can input hours, minutes, and seconds.
* **Apple-Inspired UX:** Minimal styling, smooth hover states, and monospaced numbers to prevent jitter.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run locally:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
